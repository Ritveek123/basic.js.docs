Getting Started!

1. creating a canvas

To create a canvas we have a function called createCanvas() which takes 4 parameters class, width, height, display.

Class is a name for the canvas from which we can acces on javascript or in css. Then comes width and height as the

name suggests it is used to set width and height of the canvas lastly we have display it is used to put the canvas

on a div or span for now you can just set it to body which is a build in variable in basic.js.

2. changing the color of canvas

To change the color of the canvas we have a function called setBackground() which takes 2 parameters name and

the color. here is a sample code

let canvas = createCanvas("canvas", 400, 400, body);

setBackground(canvas, "gray");

3. creating a main gameloop

To create a game loop first we have to create a function then we have a function called loop() which take one

parameter, the name of the function we have just created.

the whole code for getting started:

let canvas = createCanvas("canvas", 400, 400, body);

function update(){

}

setBackground(canvas, "gray");

loop(update);

Creating a player

1. creating a box

To create a box we have a function called createSquare() which takes 7 parameters name, w, h, x, y, display, color.

we know the the use of name and display. w is for width, h is for height, x is for x position, y is for y position

and lastly color is to set color for box and here is the code:

let box = createSquare("box", 40, 40, 20, 90, canvas, "red");

2. loading a image and displaying it to the canvas

Displaying square isn't fun so we should display some images, To do that we have a function called loadImage()

which takes 7 parameters name, img, x, y, w, h, display. I am not going to explain name, x, y, w, h, display.

img is the path of the image that you want. here is the code

let img = loadImage(".img", "test.png", 80, 40, 40, 40, canvas);

3. replacing the image

To replace the image we have a function called replaceImage() which takes 2 parameters name, image. But here the name

is the name of the image you loaded and image is the path of the image you want to replace with. here is the code:

replaceImage(img, "test(1).png")

the whole code till now:

let canvas = createCanvas("canvas", 400, 400, body);

let box = createSquare("box", 40, 40, 20, 90, canvas, "red");

let img = loadImage(".img", "test.png", 80, 40, 40, 40, canvas);

replaceImage(img, "test(1).png")

function update(){

}

setBackground(canvas, "gray");

loop(update);

Moving our player

1. Detecting wasd key

To detect wasd keys we have a function called controller(); but firstly there is an array called state which has 4

boolen objects which all are set to false first object is set to true when w key is pressed, second is when s is

pressed, third is when a is pressed, fourth is when d is pressed. So back to controller this function set the objects

of state array. Then we should add a eventListener to do certain action as the control keys are pressed and here is the code

controller();

window.addEventListener("keydown", function (ev){

if (state[0] == true){

console.log("Hello World!");

}

})

2. moving our player

To move our player we have 2 functions called setX(); and setY(); which takes two parameters target, x or y finally just

combine the controls key detection code and this setX and setY function to move our player but we should have a variable

called velocity which is the speed of player and here is the final code till now:

let canvas = createCanvas("canvas", 400, 400, body);

let box = createSquare("box", 40, 40, 20, 90, canvas, "red");

let boxX = 20;

let boxY = 90;

let velocity = 10;

let img = loadImage(".img", "test.png", 80, 40, 40, 40, canvas);

replaceImage(img, "test(1).png")

function update(){

controller();

window.addEventListener("keydown", function (ev){

if (state[0] == true){

boxY = boxY - velocity;

setY(box, boxY);

}

if (state[1] == true){

boxY = boxY + velocity;

setY(box, boxY);

}

if (state[2] == true){

boxX = boxX - velocity;

setX(box, boxX);

}

if (state[3] == true){

boxX = boxX + velocity;

setX(box, boxX);

}

})

}

setBackground(canvas, "gray");

loop(update);

Adding a text to the screen

1. creating a text on canvas

to create a text on to the canvas we have a function called addText() which takes 5 parameters. Name, text, x,

y, display. Name is class and we know the use of it. Text is for setting the text that we want. X and y, x is for

x position and y is for y position of text. lastly we know the use of display. here is the code

let txt = addText("txt", "Hello World!", 8, 8, canvas);

2. changing the color of the text

We create a text on to the screen but if you have a dark background such as black you will not be able to see the

text because it has a default black color, so to change the color we have a function called as setTextColor()

which takes two parameters text, color. Text is the variable of text and color is the name of the color. here is the code

setTextColor(txt, "white");
