Getting Started!

1. creating a canvas

To create a canvas we have a function called createCanvas() which takes 4 parameters class, width, height, display.

Class is a name for the canvas from which we can acces on javascript or in css. Then comes width and height as the

name suggests it is used to set width and height of the canvas lastly we have display it is used to put the canvas

on a div or span for now you can just set it to body which is a build in variable in basic.js.

2. changing the color of canvas

To change the color of the canvas we have a function called setBackground() which takes 2 parameters name and

the color. here is a sample code

let canvas = createCanvas("canvas", 400, 400, body);

setBackground(canvas, "gray");

3. creating a main gameloop

To create a game loop first we have to create a function then we have a function called loop() which take one

parameter, the name of the function we have just created.

the whole code for getting started:

let canvas = createCanvas("canvas", 400, 400, body);

function update(){

}

setBackground(canvas, "gray");

loop(update);
